1. Create a collection of words and images
2. Shuffle the collection 
3. Generate a card for each word and image
4. Make the cards clickable
5. When the user clicks on a card, flip it over
6. If the user clicks on a card while another is already flipped over,
    7. Check if the cards form a pair.
        8. If they do, hide the cards and grant points
        9. If they don't match, turn them back over and prompt the user to try again
10. When all the cards are gone, the user wins!
